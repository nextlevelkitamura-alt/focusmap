# PC版デイリーサイドバー

## 目的

PC版Todoの3日カレンダーを主表示のまま残し、右上のトグルから「デイリー」サイドバーを開けるようにする。サイドバーでは選択日の予定を時系列で確認し、カレンダー上の予定を選ぶとその詳細を表示する。

## 非対象

- Daily記録・目標・セッションのTursoデータモデル変更
- Google Calendar同期・予定作成・完了保存の変更
- モバイルのボトムシート編集UIの変更
- 本番push・Cloud Runデプロイ

## 現状

- PC版Todoは`DesktopTodayPanel`で3日カレンダーを表示するが、予定クリックはモバイル用の編集シートを直接開く。
- 日付ごとの予定一覧を横に保持するデイリー表示と、その開閉状態を記憶する導線はない。

## 方針

1. 右上のパネルアイコンでデイリーを開閉し、開閉状態はブラウザー内に記憶する。
2. 十分に広い画面ではカレンダー横の固定パネル、ノートPC幅ではカレンダー上に重なる右ドロワーとして表示する。
3. 予定クリックはデイリーを開いて対象予定を選択し、既存編集はデイリー内の明示的な操作から開く。
4. デイリーには選択日の予定一覧と、選んだ予定の時刻・完了状態・種別を表示する。予定がない日は空状態にする。
