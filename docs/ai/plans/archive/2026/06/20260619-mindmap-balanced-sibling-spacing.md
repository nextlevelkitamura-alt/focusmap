# 自作マップ兄弟ノード均等配置

- Task ID: TASK-20260619-007
- Status: completed
- Created: 2026-06-19
- Completed: 2026-06-19
- Board: `docs/ai/task-board.md`

## Goal

自作マップで、同じ親に紐づく子ノードを均等な縦間隔で配置する。奇数個では中央の子ノード中心を親ノード中心に揃え、偶数個では親ノード中心が中央2つの子ノード中心の中間に来るようにする。

## Scope

- `src/lib/mindmap-model.ts`
- `src/lib/mindmap-model.test.ts`
- `docs/CONTEXT.md`

## Non-goals

- React Flow版のレイアウト変更
- ノード幅、列間隔、D&D保存、折りたたみ状態の仕様変更
- デスクトップ/モバイルで別ロジックを増やすこと

## Plan

1. 自作マップ共通モデルの表示対象ツリーから親子関係を再構築する。
2. 各親の子ノードを表示順のまま等間隔スロットへ配置する。
3. 子孫の占有高さをピッチ計算に含め、均等化による枝同士の重なりを避ける。
4. 既存のX列再配置は維持し、Y配置だけを親中心基準へ置き換える。
5. 奇数/偶数/子孫付きの回帰テストを追加する。

## Parallelization

SINGLE_CHAT。編集範囲は自作マップの共通レイアウトモデルに閉じており、React FlowやCodex取り込みとは分けるほどの独立実装単位がない。

## Verification

- ユーザー明示がないため自動テスト・lint・buildは実行しない。
- 差分確認で、React Flow版やD&D保存処理へ不要な変更が入っていないことを確認する。

## Result

`buildMindMapModel` のY配置を、親中心基準の等間隔スロットへ再配置する処理に変更した。奇数個の子では中央子ノードが親中心へ揃い、偶数個では親中心が中央2つの中間に来る。子孫サブツリーの上下占有範囲をピッチ計算に含めるため、子孫付きノードでも隣接枝との重なりを避ける。
