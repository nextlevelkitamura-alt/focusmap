必要なものだけ送る流れ Raw → Report → Summary
1. 全履歴を保管

Mac agentがCodexの会話・状態をDBへ保存。ここは監査用なので途中ログも残す。

2. Report Viewを作る

送信依頼、最終報告、質問、失敗だけを抽出。progress/status/systemは落とす。

3. 画面へ表示

履歴詳細には作業後報告を中心に表示。依頼文は折りたたみ。

4. AI要約へ送信

要約AIにはReport Viewだけ送る。rawログや画面情報は送らない。
