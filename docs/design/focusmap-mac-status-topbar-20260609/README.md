# Focusmap Mac status topbar mockups - 2026-06-09

目的: 現行マップ画面の右上ツールバーに、Mac接続状態を小さく常設する案を確認する。

## 配置

- Mac接続チップは、右上ツールバーの設定アイコンの左側に置く。
- カレンダー/ドキュメント系アイコンの右、設定ギアの左に置くことで、接続状態は常時見えるが主要ナビを邪魔しない。
- 既存の下部 `Codex看板` にあるMac状態は詳細な補助表示として残し、右上は現在状態の短い入口にする。

## 状態

### Closed

ファイル: `mockup-01-mac-status-closed.png`

小さな `Mac接続` チップだけを表示する。緑ドットとPCアイコンでオンラインを示し、通常時は設定ギアより目立ちすぎない。

### Open

ファイル: `mockup-02-mac-status-open.png`

チップから小さなポップオーバーを開く。`Mac agent`、`Codex app-server`、`Runner-01`、最終確認、任意権限の不足を一覧し、`再確認` と `設定` へ進める。

## 実装時の注意

- ポップオーバーは右上に収め、マップ操作を大きく塞がない。
- 設定アイコンは既存位置のまま残す。
- オンライン/オフライン/権限不足は色だけでなく短い文言も併記する。
